\# Анализ пользователей Яндекс Афиши



\## Описание проекта

Анализ поведения пользователей платформы Яндекс Афиша для выявления факторов, влияющих на возвращаемость клиентов.



\## Структура проекта

\- `main.ipynb` - основной анализ

\- `requirements.txt` - зависимости

\- `config.py` - конфигурация подключения

\- `.env` - доступы



\## Установка и запуск

1\. Клонировать репозиторий

2\. Установить зависимости: `pip install -r requirements.txt`

3\. Создать файл `.env` с параметрами подключения

4\. Запустить Jupyter: `jupyter notebook main.ipynb`



\## Используемые технологии

\- Python 3.9+

\- Pandas, NumPy для анализа

\- SQLAlchemy для работы с БД

\- Matplotlib, Seaborn для визуализации

\- Phik для корреляционного анализа
